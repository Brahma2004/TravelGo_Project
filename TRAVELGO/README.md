# TravelGo: A Cloud-Powered Real-Time Travel Booking Platform Using AWS

## 📌 Project Overview

TravelGo is a cloud-powered web application that allows users to search, explore, and book travel services such as buses, trains, flights, and hotels in real time. The platform provides a simple and user-friendly interface for booking travel tickets and managing reservations.

The application is developed using **Python Flask for the backend** and **HTML/CSS for the frontend**, providing dynamic content and a seamless user experience.

The goal of this project is to demonstrate how cloud-based technologies can be used to build scalable and efficient travel booking systems.

---

## 🚀 Features

* 👤 User Registration and Login System
* 🚌 Bus Ticket Booking
* 🚆 Train Ticket Booking
* ✈️ Flight Ticket Booking
* 🏨 Hotel Reservation System
* 💺 Seat Selection Interface
* 💳 Secure Payment Page Simulation
* 🎫 Ticket Generation after Booking
* 📊 User Dashboard for Managing Bookings

## ☁️ AWS Services Used (Proposed Deployment)

The system can be deployed using AWS cloud services such as:

* **Amazon EC2** – Hosting the web application
* **Amazon S3** – Storing static files and project assets
* **Amazon DynamoDB / MongoDB Atlas** – Database management
* **AWS IAM** – Secure access management
* **AWS CloudWatch** – Monitoring application performance

These services ensure scalability, reliability, and secure cloud deployment.
