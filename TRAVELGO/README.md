# TravelGo – Cloud-Based Travel Booking System

TravelGo is a web-based travel booking platform that allows users to book **bus, train, flight, and hotel services** in one place.  
The application is built using **Flask (Python)** and integrates **AWS services like DynamoDB and SNS** for cloud-based data storage and notifications.

---

## 🚀 Features

- User Registration and Login
- Dashboard for managing bookings
- Bus booking
- Train booking
- Flight booking
- Hotel booking
- Seat selection interface
- Payment confirmation (demo)
- Ticket generation
- Booking cancellation
- AWS DynamoDB for storing users and bookings
- AWS SNS for sending booking confirmation notifications

---

## 🛠 Technologies Used

- Python
- Flask
- AWS DynamoDB
- AWS SNS
- HTML
- CSS
- JavaScript
- Boto3 (AWS SDK for Python)
